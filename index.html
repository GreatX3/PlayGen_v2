Project Overview

We present GameVerse, a full-stack framework for interactive video game generation, covering the entire pipeline from high-fidelity game simulation, multi-modal game generation to multi-granularity game editing.

AAA-level Game Simulation

We design a highly-compressed, low-latency 3D-VAE coupled with a rolling KV-cache latent diffusion model, achieving real-time 1080p/60 fps simulation in mainstream 3D games with robust mechanism preservation.
