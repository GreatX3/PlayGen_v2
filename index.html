Project Overview

We present GameVerse, a full-stack framework for interactive video game generation, covering the entire pipeline from simulation and generation to editing.

Specifically, GameVerse comprises three core modules: High-Fidelity Game Simulation, Multi-Modal Game Generation, and Multi-Granularity Game Editing. Collectively, GameVerse pushes interactive video game generation toward an open, extensible, and truly AI-driven creation paradigm.
